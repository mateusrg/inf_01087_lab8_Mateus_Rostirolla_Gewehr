# Lab 8 - Uso Consciente de LLMs e Prompt Engineering

## Descrição da Atividade

A atividade consiste fundamentalmente em elaborar prompts simples e detalhados, enviá-los a uma LLM, coletar suas respostas e analisá-las em diferentes contextos, visando aprender mais sobre engenharia de prompt e refletir sobre o uso adequado de IAs.

## AI Usage Statement

Este trabalho utilizou uma ferramenta de IA generativa para auxiliar na geração de explicações e exemplos de código.

Todos os prompts utilizados estão documentados em prompts.txt.

As respostas foram revisadas, testadas e verificadas pelo autor antes de serem incluídas no relatório.
